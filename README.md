# 俄罗斯方块
## 游戏平台：
- Android

## 游戏类型：
- 休闲益智

## 简介：
  - 传统的俄罗斯方块游戏，画面简洁。
  - 通过移动旋转下落的方块，将某一行填满来获取分数
  - 学习自定义View时制作，用于掌握Android的View重绘.
  - 应用了MVC设计模式。

## 使用语言
- Java

## 使用IDE：
- Android Studio
